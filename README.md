Este proyecto corresponde a la Actividad 1 de Backend con FastAPI del curso Estructuras de Datos y Algoritmos 1. La aplicación consiste en una API REST para la gestión de libros de una biblioteca universitaria. Permite registrar libros, consultarlos, actualizarlos, eliminarlos y realizar el préstamo de ejemplares disponibles.

La API fue desarrollada con FastAPI y Pydantic, aplicando validaciones sobre los datos de entrada, manejo básico de errores y documentación automática mediante Swagger en la ruta `/docs`. Como mecanismo de persistencia se utilizó una lista en memoria, por lo que la información se conserva mientras el servidor está en ejecución.

Además de los endpoints básicos del CRUD, se implementó el endpoint de préstamo y el endpoint adicional `LEYENDO`, que permite registrar los libros que una persona está leyendo actualmente.

Este trabajo fue realizado de manera individual por autorización del docente.
